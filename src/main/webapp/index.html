Java Secure Web Starter

This starter application helps you develop a secure Liberty application to avoid introducing common security vulnerabilities such as Cross-Site Scripting (XSS), Injection, Directory Traversal, and Session Hijacking.

The secure GoodServlet.java in the package employs a number of good coding practices such as input validation, output encoding, secure Http Header settings, Content Security Policy, session management, and safe SQL statement. Using it as a development starting point, your application will inherit the necessary defense against many common security vulnerabilities.
